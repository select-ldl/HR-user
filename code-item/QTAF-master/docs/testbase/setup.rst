使用前准备
==========

=================
Python版本依赖
=================

QTAF兼容python2.7以上和python3.6以上版本，windows下推荐使用ActivePython版本。Mac系统通常内置了python2.7。

=================
通过pip工具安装
=================

安装方法如下::
   
    pip install qtaf
    
.. warning:: 如果是mac os的用户，在使用pip安装的时候最好指定--user选项，否则安装可能失败或者script脚本无法使用。

===================
通过Git拉取最新代码
===================

操作方式如下::
    
    git clone https://github.com/tencent/qtaf
