手机连接电脑的步骤：
1.硬件状态正常。
        包括Android设备处于正常开机状态，USB连接线和各接口完好。
2.Android设备的开发者选项开启、USB调试模式开启。
        可以到【设置】-【开发者选项】-【Android 调试】查看。
        如果在设置例找不到开发者选项，那需要通过一个彩蛋来让它显示出来：在【设置】-【关于手机】连续点击【版本号】7次。
3.设备驱动状态正常。
        右键【计算机】-【属性】，到【设备管理器】里查看相关设备上是否有黄色感叹号或问号，如果没有就说明驱动状态已经好了。否则可以下载一个手机助手类程序（360手机助手）来安装驱动。

以上3点都确保OK后，执行命令

adb version 查看adb的版本
adb devices 查看adb连接的设备

如果连接失败：
1、下载安装360手机助手（电脑+手机）
2、手机连接电脑，usb选项选择MIDI模式
3、更换数据线，有可能线的问题
